# PyProject - Voice Assistant

This is a simple Python voice assistant (Jarvis) built using:
- `speech_recognition`
- `pyttsx3`
- `pyjokes`
- `webbrowser`

### Features:
- Voice-controlled commands
- Tells jokes
- Plays music from a custom dictionary
- Opens websites
